波形（上段） — Oscilloscope

横軸＝時間　縦軸＝振幅（音の強さ）。中央の線がゼロ（無音）。

ピッキングの強さ・一貫性
弾いた瞬間に波が大きく飛び出す。何発か弾き比べて山の高さが揃っているほど、ピッキングが安定している。
サスティン
弾いた後に波形がどれだけ続くか。コンプのアタック／リリース調整や、セッティングによるサスティンの変化確認に使える。
! クリッピング（歪みすぎ）
波形の山が水平に切れていたらクリップ中。GAINを下げるか、ペダルの出力レベルを落とす。
ノイズ・ハム
無音時に波形がザラついていればノイズフロアが高い。アース不良・シールド問題、またはアクティブPU（EMG）のバッテリー消耗を疑う。

周波数スペクトラム（下段） — FFT

横軸＝周波数（左が低音→右が高音）　縦軸＝音量（dB）。山が高いほどその帯域が強く出ている。右上のボタンで LOG＋弦ガイド と LINEAR（旧） を切替。

● 黄色マーカー＝最も音量が大きい山（多くの場合は倍音）。▼ f0 オレンジの縦線＝実際に鳴っている基音（MPM／基音検出で取得）。6E〜1E 水色の縦線＝各開放弦の基音位置（LOGモードのみ）。
※開放弦を弾くと黄色（倍音）とオレンジ（基音）が別の山に出ることがあります。チューニングの目安はオレンジ f0 と水色ガイドの一致を見てください。

④ Before / After — 録音して試聴
START 中に録音すると、現在の解析と目標トーンに基づいて "AFTER" を生成します（最大20秒）。
※ AFTER は EQ（5バンド）／ローカット（TIGHT）／コンプ（SUSTAIN） による「提案の方向」のシミュレーションです。アンプ／キャブ／IR の入れ替えや、歪み量（GRIT）の変化は再現しません。BEFORE/AFTER はラウドネスを揃えて再生します。
使い方: START → 目標を選ぶ/入力 → 数秒間しっかり弾く。鳴っている音を解析して目標との差を具体的なツマミ操作に翻訳します。 精度のコツ: マイク/部屋/IRで絶対値はブレるので、まず「基準」を保存して差分で追うと信頼度が上がります。提案は方向のヒントであって絶対値ではありません。DI（IN セレクタ）入力だと精度が上がります。
