進行を作る

コードを選んで「＋追加」すると、ここに進行が並びます。
90
使い方
・上で ルート音（C, C#…）と 種類（major, m7…）を選ぶと、中央に現在のコードが出ます。
・キーを選ぶと、そのキーのダイアトニックコード（曲づくりで軸になる7つのコード）が緑枠で光り、 コードや進行に ディグリー（I, ii, V7 など）が表示されます。進行をディグリーで覚えると、どのキーにも移調できるようになります。
・🎵 ストラムでその場で試聴。＋追加で進行に書き込み。
・進行のコードは −/＋ で長さ（拍数）を変更、右上の × で削除。ドラッグで並べ替え。
・スタイル：どんな編成で鳴らすかを選択（ギターのみ／ポップ・バラード／ロック／ファンク／シティポップ／ボサノバ／ジャズ 等）。ドラム・ベース・ピアノ・ギターが自動で伴奏します。選ぶと推奨テンポに切り替わります。
・🥁🎸🎹🎼 のチェックで、鳴らす楽器を個別に ON/OFF できます（ギターだけ、ドラム抜き 等）。
・音はすべてブラウザ内で合成（ギター＝Karplus-Strong 弦モデル、ピアノ＝FM電子ピアノ、ドラム/ベース＝シンセ）。読み込み無し＝途切れません。
・進行はこの端末に自動保存され、URL を共有すれば同じ進行を再現できます。
すべてブラウザ内で音を合成（外部ファイル・通信なし）。ネットが不安定でも止まりません。
iPhone / iPad は最初に一度どこかをタップすると音が出ます。

🥁 DRUM MACHINE — オフライン・ステップシーケンサー

110
0
使い方・キーボード
・各マスをタップで音を ON / OFF。もう一度押すと OFF。
・濃いマスは拍の頭（4つごと）。再生位置は青枠でハイライト。
・楽器名をタップでその音を試聴。M=ミュート S=ソロ（各行のボタン）。
・Space 再生/停止、C クリア、R ランダム。
・右クリック / 長押しでアクセント（強打・白枠）。
・パターンは自動でこの端末に保存され、次回開いたとき復元されます。
・URL の # 以降にパターンが入るので、その URL を共有すれば同じパターンを再現できます。
音源はすべてブラウザ内で合成（外部ファイル無し）。ネット接続が無くても途切れず動きます。
iPhone / iPad は最初に画面のどこかを一度タップすると音が有効になります。
